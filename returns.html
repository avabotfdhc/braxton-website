Returns & Refunds

Our hassle-free return policy

Return Policy

We want you to be completely happy with your purchase! If you're not satisfied, we accept returns within 30 days of delivery.

Items Eligible for Return

  • Clothing that is unworn, unwashed, with original tags attached
  • Hats and headwear in original condition
  • Accessories in original packaging, unused
  • Plush toys that are unopened and in original packaging

Items NOT Eligible for Return

  • Worn, washed, or damaged items
  • Items without original tags or packaging
  • Personalized or custom items
  • Items marked "Final Sale"
  • Gift cards

How to Return

  1. Contact us at hello@bwildandwonder.com with your order number and reason for return
  2. Receive return authorization with instructions and return address
  3. Ship the item back to us (customer pays return shipping unless item is defective)
  4. Refund processed within 5-7 business days of receiving your return

Refund Options

  • Original payment method — Refund to the card or account used for purchase
  • Store credit — Full amount including original shipping, never expires

Damaged or Defective Items

If you receive a damaged or defective item, contact us within 7 days. We'll send a replacement at no charge or issue a full refund, including return shipping.

Exchanges

For size exchanges, return the original item for a refund and place a new order for the correct size. This ensures you get the size you need quickly!
